# psnvm

```psnvm [option...] source```

## psnvm source-filename arguments...

Run source file.

## psnvm -c out-filename source-filename...

Compile source files and output validated binary.

## options

|option|params|stage|details|
|--|--|--|--|
|-z||safe vm|セーフVMを使用し、不正なメモリアクセスに対して常に 0 を返す|
