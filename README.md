# libamtrs

This repository have a two projects.

|Project|Note|
|--|--|
|libamtrs|multiplatform library for c++.|
|psnvm|c++ compatibile script language.|

## Build

1. windows

```sh
git clone https://github.com/isaponsoft/libamtrs.git
cmake libamtrs.git
MSBUILD.exe ALL_BUILD.vcxproj
```

2. other os

```sh
git clone https://github.com/isaponsoft/libamtrs.git
cmake libamtrs.git
make
```
